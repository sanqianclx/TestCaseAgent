# TestGenerate Agent - 项目完成总结

> **更新时间**: 2026-06-11  
> **项目版本**: V3.1

---

## 🎯 项目定位

**TestGenerate Agent** 是一个基于 Mastra + DeepSeek LLM 的智能测试生成系统，支持 Python、Java、C++ 三语言自动化测试生成，具备自愈循环、真实覆盖率统计、Web 界面和自主 Agent 模式。

## ✅ 已完成功能

### 🏗️ 核心架构

| 模块 | 实现状态 | 说明 |
|------|---------|------|
| **7 个 Agent** | ✅ | 测试用例生成、测试代码生成(标准+Pro)、失败诊断(标准+Pro+决策)、CLI对话 |
| **核心工作流** | ✅ | `generate-test-workflow.ts` - 7 步：解析→用例→导出计划→代码→执行→自愈→导出 |
| **7 个工具** | ✅ | read/write/parse/execute/coverage/export/logger |
| **3 个语言适配器** | ✅ | Python(pytest+coverage.py)、Java(JUnit5+Maven+JaCoCo)、C++(GoogleTest) |
| **自主 Agent 模式** | ✅ | `--autonomous` 交互式工具调用（9 个工具），支持 shell 审批机制 |

## 🚀 使用方式

### 1. CLI 命令行模式

```bash
npm run generate -- --input ./src/example.py --output ./my-tests
```

### 2. CLI 交互模式

```bash
npm run generate -- --interactive
```

### 3. 自主 Agent 模式

```bash
npm run agent
```

### 4. Web 界面

```bash
npm run server:dev      # 后端 (http://localhost:3000)
cd client && npm run dev  # 前端 (http://localhost:5173)
```

### 5. Mastra Studio 调试

```bash
npx mastra dev
# 访问 http://localhost:4111
```

## 🛠️ 技术栈

### 后端
- **框架**: Express + TypeScript
- **ORM**: Prisma
- **数据库**: MySQL 8.0
- **认证**: JWT + bcrypt
- **校验**: Zod
- **Agent 框架**: Mastra
- **LLM**: DeepSeek（v4-flash 快通道 + v4-pro 慢通道）

### 前端
- **框架**: React 18 + TypeScript
- **构建**: Vite
- **UI**: Ant Design
- **路由**: React Router
- **状态**: Zustand
- **请求**: Axios

### Python 运行时
- **解析**: Python `ast` 模块
- **测试框架**: pytest
- **覆盖率**: coverage.py

### Java / C++ 解析
- **解析器**: tree-sitter（tree-sitter-java / tree-sitter-cpp）

### Java 支持
- **测试框架**: JUnit 5
- **构建**: Maven
- **覆盖率**: JaCoCo

### C++ 支持
- **测试框架**: GoogleTest
- **编译器**: g++ (C++17)
- **覆盖率**: 预留（gcov / OpenCppCoverage）
